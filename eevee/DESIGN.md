# 이브이 농장 - 기획서

## 게임 개요
쿼터뷰 이브이 농장 시뮬레이션. 탐색을 통해 아이템을 수집하고, 진화 조건을 충족시켜 모든 이브이 진화형(17종)을 모으는 게임.
- 캔버스: 480x720 (세로 모드)
- PC(마우스) / 모바일(터치) 동시 지원

## 핵심 루프
```
농장 화면 (낮/밤)
 → 탐색 (스태미나 1칸 소모, 열매+아이템 획득)
 → 이브이 터치 (열매·아이템 전달, 친밀도/스태미나 변화)
 → 가방 (아이템 확인, 버리기)
 → 힌트 (턴당 1회)
 → 휴식 (밤↔낮 전환, 스태미나 풀충전, 진화 체크)
 → 반복
```

---

## 스태미나 시스템
- 최대 3칸 (소수점 관리)
- 탐색 1회 = 스태미나 1칸 소모
- **자동 회복**: 농장에서 패널/팝업이 없을 때 프레임마다 회복
  - 낮: 0.0003/프레임 (~100초에 1칸)
  - 밤: 0.0005/프레임 (~60초에 1칸)
- 휴식 시 풀충전
- 진화 완료 후 풀충전

## 밤낮 시스템
- 휴식 버튼으로 낮↔밤 전환
- 전환 시: 스태미나 풀충전, 탐색지 재배치, 힌트 초기화
- 휴식 시 진화 조건 체크 (조건 충족 시 진화 발생)

## 탐색 시스템
- 휴식할 때마다 이용 가능한 탐색지 중 3곳을 랜덤 제공
- 탐색 1회 = 스태미나 1칸 소모
- 드롭: 열매와 아이템 독립 판정
- 낮 전용 / 밤 전용 / 아무때나 탐색지 구분
- 스태미나 부족 시 "이브이가 지쳐있다..." 메시지

| 화산 | 낮 | 불 | 불꽃의돌, 태양조각 |

### 후반 탐색지 (해금형)

| 탐색지 | 시간 | 해금 조건 | 아이템 드롭 |
|---|---|---|---|
| 거대소굴 | 아무때나 | 기본 8종 클리어 | 다이버섯(희귀), 태양조각, 달조각 |
| 고대유적 | 밤 | 다이브이 보유 | 이브이나이트(희귀), 달조각 |
| 용의계곡 | 낮 | 다이브이+메가브이 보유 | 용의비늘(희귀), 태양조각 |

> 희귀 진화석(다이버섯, 이브이나이트, 용의비늘)은 드롭 확률 감소 (weight 15)

### 드롭 확률 시스템
- **아이템**: 40% 확률로 장소별 아이템 1개 (가중치 랜덤)
- **열매**: 60% 확률로 전체 열매 중 1개 (6종 균등)
- **보장**: 둘 다 안 나오면 열매 1개 보장
- 획득 아이템은 농장 바닥에 배치 → 터치하여 회수

## 친밀도 시스템 (현재 이브이)
- 이브이마다 좋아하는 열매 1개, 싫어하는 열매 1개 랜덤 세팅
- 유저는 처음에 모르는 상태 → 먹여보고 반응으로 판별
  - 좋아함: ♥ (분홍) + 변화량 표시
  - 싫어함: 💢 (빨강) + 변화량 표시
  - 보통: ⚡ (스태미나 회복)
- 발견된 좋아함/싫어함은 인벤토리·팝업에서 초록/빨강 테두리로 표시

| 항목 | 수치 |
|---|---|
| 시작 친밀도 | 3 |
| MAX (진화 가능) | 20 |
| 좋아하는 열매 | +3 |
| 보통 열매 | 친밀도 변화 없음, 스태미나 +1 |
| 싫어하는 열매 | -2 |
| 태양조각 (낮에 줄 때) | +5 |
| 태양조각 (밤에 줄 때) | 변화 없음 |
| 달조각 (밤에 줄 때) | +5 |
| 달조각 (낮에 줄 때) | 변화 없음 |

### 즉시 진화 체크
- 싫어하는 열매로 친밀도 0 이하 도달 시 → 즉시 고스트 진화 체크
- 태양/달 조각으로 친밀도 MAX 도달 시 → 즉시 에브이/블래키 진화 체크
- 아이템 전달 시 → 즉시 진화 체크

---

## 활동성 시스템 (농장 진화이브이)
- 수집된 진화이브이 각각 활동성(0~50)과 좋아하는/싫어하는 열매를 가짐
- 바닥의 열매에 접근하면 자동 섭취 → 활동성 변화
  - 좋아하는 열매: +5
  - 보통 열매: +2
  - 싫어하는 열매: -3
- **활동성 최대(50)인 이브이는 더 이상 자동 섭취하지 않음**
- 활동성이 이동 속도에 영향 (0.0002 ~ 0.0015)
- 활동성이 높으면 걷기 많음, 낮으면 잠자기/가만히 있기 많음

## 진화 조건

### 진화 체크 타이밍
- 아이템 전달 → 즉시
- 친밀도 0 이하 도달 → 즉시
- 친밀도 MAX + 태양/달 조각 → 즉시
- 휴식(턴 전환) 시 → 자동

### 진화 후 흐름
1. 진화 연출 (이브이 → 진화형, 약 2초)
2. 진화한 이브이는 농장에 배치 (활동성·좋아함/싫어함 새로 세팅)
3. 새 이브이가 아래에서 걸어 들어옴 ("안녕!" 말풍선)
4. 새 이브이의 친밀도 3, 좋아함/싫어함 랜덤 재세팅
5. 밤↔낮 전환, 스태미나 풀충전, 탐색지 재배치

### 기본 8종

| 진화형 | 타입 | 조건 |
|---|---|---|
| 샤미드 | 물 | 물의돌 전달 |
| 쥬피썬더 | 전기 | 천둥의돌 전달 |
| 부스터 | 불 | 불꽃의돌 전달 |
| 에브이 | 에스퍼 | 친밀도 MAX + 낮 |
| 블래키 | 악 | 친밀도 MAX + 밤 |
| 리피아 | 풀 | 이끼바위 제작(조각x3) + 전달 |
| 글레이시아 | 얼음 | 얼음바위 제작(조각x3) + 전달 |
| 님피아 | 페어리 | 친밀도 MAX + 리본 전달 |

### 팬메이드 9종 (기본 8종 클리어 후 도감 칸 등장)

| 진화형 | 이름 | 조건 |
|---|---|---|
| 왕 | 다이브이 | 다이버섯 전달 (친밀도 ≥ 10 필요) |
| 메가 | 메가브이 | 이브이나이트 전달 (친밀도 ≥ 10 필요) |
| 격투 | 파이티아 | 탐색 20회 누적 + 친밀도 ≥ 10 |
| 고스트 | 레이시아 | 친밀도 0 이하로 추락 |
| 드래곤 | 드라키온 | 다이브이+메가브이 보유 + 용의비늘 전달 (친밀도 ≥ 15) |
| 땅 | 샌디온 | 총 휴식 8회 이상 |
| 바위 | 록키온 | 같은 탐색지 연속 5회 방문 + 친밀도 ≥ 8 |
| 비행 | 에어리아 | 모든 탐색지(해금 포함) 1회 이상 방문 |
| 강철 | 메탈리아 | 진화돌 3종(물·불·천둥) 동시 보유 상태에서 휴식 |

> 왕/메가/드래곤 진화석은 친밀도 조건 미달 시 "아직 준비가 안 된 것 같다..." 메시지로 전달 거부 (아이템 소모 안됨)

### 바위재료 힌트
- 이끼바위조각/얼음바위조각 첫 획득 시 제작 힌트 메시지 표시

## 아이템 관리

### 인벤토리 (가방)
- 보유 아이템 목록 + 좋아함/싫어함 라벨 표시
- **버리기만 가능** (전달 기능 없음)
- 버리기: 인벤에서 제거 → 농장 바닥에 물리 오브젝트로 배치
- 스크롤 지원 (터치 드래그 / 마우스 휠)

### 아이템 전달 (이브이 터치 팝업)
- 현재 이브이를 터치하면 전달 팝업 표시
- 팝업 상단에 친밀도·스태미나 수치 표시
- **열매**와 **아이템** 섹션으로 구분 표시
- **열매 전달**: 팝업 유지 (연속 전달 가능)
- **아이템 전달**: 팝업 닫힘 (진화 결과 확인)

### 농장 바닥 아이템
- 탐색으로 획득한 아이템이 바닥에 배치됨
- 터치하여 인벤토리로 회수
- 이브이들이 지나가며 밀어냄 (물리 시뮬레이션)
- 농장 진화이브이가 열매 자동 섭취 → 활동성 변화

### 제작 (인벤토리에서)

| 아이템 | 재료 |
|---|---|
| 이끼바위 | 이끼바위조각 x3 |
| 얼음바위 | 얼음바위조각 x3 |

## 힌트 시스템
- 턴당 1회 사용 가능 (휴식 시 초기화)
- 스태미나 소모 없음
- 수집하지 않은 진화형에 대한 힌트를 랜덤 제공
- 포스트게임 힌트는 기본 8종 클리어 후 등장
- 모든 진화 완료(17종) 후 힌트 기능 소멸

---

## 농장 화면
- 쿼터뷰 배경 (다이아몬드형 바닥)
- 밤/낮 전환 시 배경색·하늘 변화 + 배경 캐시 갱신
- 현재 이브이: 머리 위 친밀도 바 표시 (하늘색, MAX시 분홍)
- 농장 진화이브이: 머리 위 활동성 바 표시 (주황색 계열)
- 모든 이브이 AI: idle(가만히) / walk(걷기) / sleep(수면) 상태 전환
- 밤에는 수면 확률 증가
- 바닥 아이템 물리 시뮬레이션

## UI 구성
- **상단 HUD**: 밤/낮 아이콘, 수집 현황 (N/17)
- **하단 버튼**: 탐색, 가방, 힌트, 도감, 휴식
- **이브이 터치 팝업**: 열매/아이템 구분, 친밀도·스태미나 수치
- **농장 이브이 터치 팝업**: 타입, 활동성 바, 열매 전달
- **탐색 선택 패널**: 현재 턴의 탐색지 목록 (낮밤·속성 표시)
- **탐색 결과 팝업**: 장소 미니뷰 + 진행 바 + 결과 아이템 표시
- **가방 패널**: 제작 버튼 + 아이템 목록 (버리기)
- **도감 패널**: 기본 8칸 + 포스트게임 9칸 (미수집=?, 수집=스프라이트+이름)
- **패널 닫기**: 바깥 터치

- 기능:

  - 친밀도 MAX / 친밀도 → -1
  - 스태미나 MAX
  - 탐색/휴식 카운트 충전
  - 기본 8종 해금 (포스트게임 진입)

## 엔딩
- 17종 전부 수집 시:
  1. 축하 화면 ("★ 축하합니다! ★")
  2. 전체 이브이 퍼레이드 애니메이션
  3. 통계 표시 (총 탐색 횟수, 총 휴식 횟수, 턴 수, 수집 수)
  4. 터치하여 농장으로 복귀
- 일반 이브이 1마리 추가 등장
- 이후 감상 모드: 18마리(진화 17종 + 이브이 1) + 굴러다니는 아이템 = 힐링 농장
