# 이브이 농장 - 기획서 v6

## 게임 개요
쿼터뷰 이브이 농장 디펜스. 농장에 침입하는 울트라비스트를 이브이들이 자동으로 요격하고, 탐색으로 진화 재료를 모아 전력을 강화하는 순환 구조 게임.
- 캔버스: 480x720 (세로 모드)
- PC(마우스) / 모바일(터치) 동시 지원
- 슬로건: "이브이 농장을 지켜주세요!"

## 핵심 컨셉
```
농장에 몬스터가 상시 침입
  → 농장의 이브이들이 자동으로 공격 (DPS)
  → 제한시간 내 처치 실패 시 게임오버
이브이 진화 = DPS 강화 수단
탐색 = 진화 재료 획득 (이브이 파견 → 농장 DPS 감소)
전략 = 탐색 vs 전투 인원 배분
```

## 전체 플로우
```
게임 시작 (닉네임 입력)
  → 농장에 몬스터 등장 (자동 전투)
  → 탐색 출동 (이브이 파견 → 대기 → 귀환, 열매+아이템)
  → 열매로 이브이 활동성↑ → DPS↑
  → 진화 아이템으로 이브이 진화 → 타입 능력 획득
  → 몬스터 처치 → 다음 몬스터 (더 강함) / 스킵 버튼으로 즉시 소환 가능
  → 도감 진행에 따라 새 탐색지 해금
  → 제한시간 초과 → 게임오버 (리셋, 리더보드 등록)
  → 19종 컴플리트 → 엔딩 + 활동성 무한 성장
  → 무한 스테이지 도전
```

## 1. 전투 시스템 (농장 상시 전투)

### 기본 구조
- 농장에 몬스터가 항상 존재 (배회하며 열매를 공격)
- 농장에 있는 이브이들이 몬스터를 향해 자동 돌진 + 공격
- 탐색 파견 중인 이브이는 농장에 없음 → DPS 감소
- 제한시간 내 처치 실패 → 게임오버

### DPS 계산
```
이브이 DPS = basePower + activity × 0.3
  × 상성 배율 (다중 타입)
  × 고유 능력 보정
```

| 이브이 | basePower |
|---|---|
| 기본 이브이 | 0.5 |
| 일반 진화 | 1.0 |
| 거다이브 (노멀) | 1.8 |
| 메가브이 (노멀) | 2.2 |
| 컴플리트 후 | +0.5 보너스 |

### 열매 HP 시스템
- 열매 HP: **2** (고정)
- 몬스터가 열매에 충돌 시 공격 (항상 1 데미지)
- 공격 쿨다운: 0.75초
- 충돌 시 열매가 밀려남 (물리 밀어내기)
- 1회 타격: 반짝이 이펙트 제거 + 깜빡임 표시
- 2회 타격(파괴): 파편 파티클 + 이모지 페이드 이펙트

### 몬스터 행동
- 농장을 배회하며 열매를 공격 (다회 타격으로 파괴, 파괴 시 최대HP 10% 회복)
- 이동속도: 원작 스피드 기반 × 1.2배 (보스는 0.7배 감속)
- 이브이를 밀어냄 (넉백 시스템, 쿨다운 있음)
  - 모든 몬스터: 반경 0.14, 넉백 0.07, 경직 0.5초
  - 쿨다운: 보스 3초, 일반 2초
  - 넉백 중 AI 정지, 관성 감속 (0.88/프레임)
  - 넉백된 이브이가 근처 이브이도 연쇄 밀어냄 (속도 60% 전달)
- 등장: 하늘에서 떨어지는 연출 + 먼지 이펙트
- 처치: 폭발 파티클 + 페이드아웃 + 스킵 버튼

### 몬스터 HP
```
일반: 30 + stage × 25 + stage^1.6 × 5
보스: 일반 × 3
```

### 제한시간 (실시간 Date.now 기반)
```
기본: 60초 + 보스클리어 수 × 5초
보스: 기본 × 1.5
스테이지 진행으로는 증가하지 않음 (보스 클리어 시에만 +5초)
초과 시 게임오버
탐색/전투 모두 실시간 기반 (모니터 주사율 무관)
```

## 2. 탐색 시스템

### 출정 방식
```
탐색 버튼 → 턴당 3곳 랜덤 제공 → 장소 선택
  → 이브이 선택 (4열 그리드, 토글 선택, 협동 파견 가능)
  → "출동!" 버튼 → 파견 (실시간 대기)
  → 자동 귀환 + 결과 팝업 + 탐색지 재셔플
  → 귀환 시 활동성 +1 (님피아 포함 시 전원 +3), 위치 랜덤 재배치
```

### 협동 파견
| 도감 수집 | 협동 인원 |
|---|---|
| 0~7 | 1마리 |
| 8+ (기본 8종) | 2마리 |
| 14+ | 3마리 |

- 협동 시간 보너스: 2마리 -30%, 3마리 -50%
- 드롭: 각 이브이별 독립 판정 (능력도 각각 적용)
- 활동성 소모: 파견 시 -3 (님피아 포함 시 전원 소모 없음)

### 탐색 규칙
- 파견 슬롯: 도감 5종→2, 8종→3, 12종→4 (최대 4)
- 파견 중 이브이: 농장에서 사라짐 (DPS 감소, 전투 불참)
- 드롭: 열매 → 농장 바닥, 아이템 → 가방 직행
- 탐색 최대시간: 몬스터 제한시간의 80% 이내로 cap
- 가속장치 사용: 남은 시간 50% 감소
- 귀환 시 친밀도 +1

## 3. 이브이 시스템

### 이브이 둥지
- 기본 이브이 0마리일 때 소환 가능
- 비용: 열매 `3 + 도감 수 × 3`개 (점진 증가)
- 진화 시 자동 추가 X → 수동 소환

### 능력치
| 수치 | 역할 |
|---|---|
| **활동성** (activity) | DPS 기반, 이동속도, 탐색 효율. 열매/조각으로 증가. **모든 이브이 공통.** |
| **친밀도** (friendship) | **진화 조건 전용** (기본 이브이만). 좋아함/싫어함 열매로 변화. |
| **속도** (spd) | 진화 이브이별 고유 속도. 기본(55) 초과분이 이동속도 보너스로 적용. |

### 활동성 상세
- 초기: 15
- 상한: 50 (컴플리트 후 무제한)
- 이동속도: 활동성 50 기준 cap + 진화 spd 보너스
- 열매 효과:

| 열매 | 활동성 변화 |
|---|---|

| 싫어하는 열매 | -3 |

| 이브이 | 효과 |

- 활동성 MAX일 때: 바닥 열매/아이템 자동 섭취 차단

> 친밀도 MAX 달성 시 즉시 진화 체크 (밤낮 전환 대기 불필요)

## 5. 아이템

### 전달 규칙
- 진화 아이템만 이브이에게 전달 가능
- 태양/달 조각: 기본 이브이 + 진화 이브이 모두 전달 가능 (활동성↑)

- 탐색 패널에서 진행 중 탐색에 사용 → 남은 시간 50% 감소

## 7. UI

### 상단 HUD
- 밤/낮 아이콘 + 경과 바 + 남은 시간
- 전투 인원 ⚔N / 탐색 인원 🏃N
- 도감 N/8 (기본) 또는 N/19 (포스트게임)
- 탐색 출정 프로그레스 바 (파견 중일 때)

### 하단 버튼 (균등 배치)
탐색 | 가방 | 힌트 | 도감 | 🏆랭킹

### 팝업 (아이템 전달은 화면 중앙 배치)
- 이브이 전달 (열매/아이템 구분, 열매 전달 시 팝업 유지)
- 진화이브이 전달 (열매 + 태양/달 조각)
- 탐색 결과 (이브이 스프라이트 + 드롭 목록)
- 처음부터 확인 (세이브 삭제)
- 치트 메뉴
- 닉네임 입력
- 리더보드 (로딩 → 상위 10명)

### 도감 (상세 리스트)
- 스프라이트, 이름, 타입
- 고유 능력
- 현재 상태 (전투 중 / 탐색 중)
- 활동성 바
- 좋아함/싫어함 열매 이모지
- DPS

### 농장 화면
- 쿼터뷰 배경 + 실시간 밤낮 전환
- 이브이들: 몬스터를 향해 자동 돌진 + 공격, 이브이 간 충돌 밀어내기
- 몬스터: 스프라이트 + HP바 + 이름/타입(아웃라인) + 타이머 + DPS
- 바닥 아이템: 터치 회수, 물리 시뮬레이션, 열매 반짝이 이펙트
- 둥지 버튼: 이브이 없을 때 하단 메뉴 위에 표시
