# 이브이 농장 - 기획서

## 게임 개요
횡스크롤 이브이 농장 시뮬레이션. 탐색을 통해 아이템을 수집하고, 진화 조건을 충족시켜 모든 이브이 진화형을 모으는 게임.

## 핵심 루프
```
턴 시작 (낮/밤) → 진화 조건 체크
 → 탐색 보내기 / 과일 주기 / 아이템 전달 / 아이템 버리기 / 힌트 / 스킵
 → 탐색 소진 or 스킵
 → 밤낮 전환
 → 반복
```

---

## 밤낮 시스템
- 탐색을 모두 소진하면 낮↔밤 전환
- 유저는 아무 행동 없이 스킵으로 전환 가능
- 전환 시 탐색 횟수 재충전
- 탐색 횟수 = 농장 이브이즈 수

## 탐색 시스템 (프린세스메이커식)
- 탐색지를 유저가 직접 선택
- 탐색 1회 = 아이템 1개 획득 (확률 기반)
- 낮 전용 / 밤 전용 / 아무때나 탐색지 구분

---

|---|---|---|---|---|---|
| 호수 | 낮 | 물 | 크라보열매, 오랭열매 | 물의돌 | 태양조각 |
| 화산 | 낮 | 불 | 치리열매, 오랭열매 | 불꽃의돌 | 태양조각 |

| 숲 | 아무때나 | 풀 | 모모열매, 나나열매 | 이끼바위조각 | 오랭열매 |
| 발전소 | 아무때나 | 전기 | 나나열매, 치리열매 | 천둥의돌 | 오랭열매 |
| 설산 | 아무때나 | 얼음 | 크라보열매, 라즈열매 | 얼음바위조각 | 오랭열매 |
| 신전 | 밤 | 에스퍼 | 라즈열매, 오랭열매 | 태양조각 | 달조각 |
| 폐탑 | 밤 | 악 | 나나열매, 라즈열매 | 달조각 | 태양조각 |

| 탐색지 | 시간 | 해금 조건 | 드롭 아이템 |
|---|---|---|---|
| 거대소굴 | 아무때나 | 기본 8종 클리어 | 다이버섯 |
| 고대유적 | 밤 | 다이브이 보유 | 이브이나이트 |
| 용의계곡 | 낮 | 다이브이+메가브이 보유 | 용의비늘 |

## 열매 시스템 (6종)

| 열매 | 맛 | 주요 출현지 |
|---|---|---|
| 오랭열매 | 무난 | 호수, 꽃밭, 신전 |
| 치리열매 | 매운맛 | 화산, 발전소 |
| 크라보열매 | 신맛 | 호수, 설산 |
| 모모열매 | 달콤 | 꽃밭, 숲 |
| 나나열매 | 떫은맛 | 숲, 발전소, 폐탑 |
| 라즈열매 | 건조 | 설산, 신전, 폐탑 |

## 친밀도 시스템
- 이브이마다 좋아하는 열매 1개, 싫어하는 열매 1개 랜덤 세팅
- 유저는 모르는 상태에서 시행착오로 파악 (이브이 반응 애니메이션으로 판별)

| 항목 | 수치 |
|---|---|
| 시작 친밀도 | 3 |
| MAX (진화 가능) | 10 |
| 좋아하는 열매 | +3 |
| 보통 열매 | +1 |
| 싫어하는 열매 | -2 |
| 태양조각 (낮에 줄 때) | +4 |
| 태양조각 (밤에 줄 때) | 0 |
| 달조각 (밤에 줄 때) | +4 |
| 달조각 (낮에 줄 때) | 0 |

## 진화 조건

### 진화 타이밍
- 모든 진화는 턴(밤/낮) 시작 시 자동 체크
- 아이템 전달, 친밀도 변화 등은 현재 턴에 수행 → 다음 턴 시작 시 진화

### 기본 8종

| 진화형 | 타입 | 조건 |
|---|---|---|
| 샤미드 | 물 | 물의돌 전달 |
| 쥬피썬더 | 전기 | 천둥의돌 전달 |
| 부스터 | 불 | 불꽃의돌 전달 |
| 에브이 | 에스퍼 | 친밀도 MAX + 낮 |
| 블래키 | 악 | 친밀도 MAX + 밤 |
| 리피아 | 풀 | 이끼바위 제작(조각x3) + 전달 |
| 글레이시아 | 얼음 | 얼음바위 제작(조각x3) + 전달 |
| 님피아 | 페어리 | 친밀도 MAX + 리본 전달 |

### 팬메이드 9종 (기본 8종 클리어 후 도감 칸 등장)

| 진화형 | 이름 | 조건 |
|---|---|---|
| 왕 | 다이브이 | 거대소굴에서 다이버섯 획득 → 전달 |
| 메가 | 메가브이 | 고대유적에서 이브이나이트 획득 → 전달 |
| 격투 | 파이티아 | 한 이브이 탐색 N회 누적 |
| 고스트 | 레이시아 | 친밀도 0 이하로 추락 |
| 드래곤 | 드라키온 | 다이브이+메가브이 보유 + 용의비늘 전달 |
| 땅 | 샌디온 | 밤낮 N회 스킵 |
| 바위 | 록키온 | 같은 탐색지 연속 3회 방문 |
| 비행 | 에어리아 | 모든 탐색지 1회 이상 방문 |
| 강철 | 메탈리아 | 진화돌 3종 동시 보유 상태에서 턴 넘김 |

### 바위재료 힌트
- 바위조각 첫 획득 시 멘트 표시
- 예: "이 이끼바위조각으로 커다란 이끼바위를 만들 수 있을 것 같다"

## 아이템 관리
- 인벤토리에 보유
- 이브이에게 전달 (진화/친밀도)
- **농장에 버리기**: 인벤에서 제거, 농장 바닥에 물리 오브젝트로 존재
  - 이브이들이 지나가며 발로 차고 굴러다님
  - 순수 비주얼 요소

### 제작

| 아이템 | 재료 |
|---|---|
| 이끼바위 | 이끼바위조각 x3 |
| 얼음바위 | 얼음바위조각 x3 |

---

## 힌트 시스템
- 탐색 1회 소모
- 현재 이브이의 진화 가능성에 대한 두루뭉실한 힌트
- 예시:
  - "이브이가 물소리에 귀를 기울이고 있다..." (물의돌 필요)
  - "이브이가 밤하늘을 유독 좋아하는 것 같다..." (블래키)
  - "이브이가 무언가 단단한 것에 관심을 보인다..." (바위/강철)
- 모든 진화 완료(17종) 후 힌트 기능 소멸

---

## 농장 화면
- 횡스크롤 배경, 밤/낮 전환 (배경색, 하늘 변화)
- 이브이들이 자유롭게 배회
- 이브이 간 랜덤 상호작용 애니메이션 (장난치기, 낮잠, 쫓아다니기 등)
- 밤이면 졸거나 웅크리는 모션
- 버린 아이템이 바닥에 굴러다님 (물리엔진)

## UI 구성
- **상단 HUD**: 밤/낮 표시, 남은 탐색 횟수
- **하단 버튼**: 탐색 가기, 아이템 가방, 과일 주기, 스킵(다음 턴)
- **탐색 선택 화면**: 탐색지 목록 (잠금/해금, 낮밤 제한 표시)
- **탐색 결과 화면**: 획득 아이템 표시
- **아이템 가방**: 보유 아이템 목록, 선택 → 전달/버리기
- **도감**: 기본 8칸(미수집=?), 8종 클리어 후 추가 9칸 등장(미수집=?)
- **진화 연출**: 턴 시작 시 조건 충족되면 자동 재생
