# 이브이 농장 - 기획서 v3

## 게임 개요
쿼터뷰 이브이 농장. 이브이를 진화시켜 전투 전력을 키우고, 전투로 새 탐색지를 열어 더 많은 진화를 만드는 순환 구조 게임.
- 캔버스: 480x720 (세로 모드)
- PC(마우스) / 모바일(터치) 동시 지원

## 핵심 컨셉
```
이브이 진화 = 전투력 확보 수단
탐색 = 진화 재료 획득
전투 = 새 탐색지 해금
탐색 ↔ 전투가 서로의 문을 열어주는 순환
```

## 전체 플로우
```
탐색 → 진화 재료 + 열매 획득
  ↓
이브이 진화 → 전투 전력↑
  ↓
전투 (이브이 선택 → 자동 배틀) → 승리
  ↓
전투 보상 → 새 탐색지 해금 / 탐색 슬롯 확장
  ↓
새 탐색지 → 더 희귀한 진화 재료 → 더 많은 진화
  ↓
반복 → 최종 보스 클리어 = 엔딩
```

## 핵심 루프
```
농장 화면 (낮/밤)
 → 탐색 출동 (이브이 파견 → 시간경과 → 귀환)
 → 이브이 관리 (열매 전달, 친밀도, 진화)
 → 전투 (스테이지 선택 → 이브이 선택 → 자동 배틀)
 → 휴식 (밤↔낮 전환, 진화 체크)
 → 반복
```

## 1. 전투 시스템 (메인 콘텐츠)

### 기본 구조
```
스테이지 표시 (적 타입·레벨 공개)
  → 이브이 1마리 선택 (유저의 전략적 선택)
  → 자동 전투 (연출)
  → 승패 판정 → 보상
```

### 승패 판정
```
내 전투력 = 기본 스탯 + 친밀도 보정 + 상성 보정
적 전투력 = 스테이지별 고정값

전투력 > 적 → 승리
전투력 ≤ 적 → 패배
```

| 요소 | 보정 |
|---|---|
| 상성 유리 | x1.5 |
| 상성 불리 | x0.6 |
| 상성 보통 | x1.0 |
| 친밀도 MAX | 추가 보정 |

### 유저 전략
- 적 타입이 공개되어 있음 → 상성 유리한 이브이 선택
- 친밀도 낮은 유리 타입 vs 친밀도 MAX 불리 타입 → 판단 필요
- 연전 시 활동성 소모 → 강한 이브이를 아끼거나 분산 운용

### 스테이지 구성 (던전형)
```
던전 1: 초원의 시련
 ├── 1R: 풀타입 Lv.3
 ├── 2R: 물타입 Lv.4
 ├── 3R: 풀+물 Lv.5
 └── 보스: ??? Lv.8
 └── 클리어 보상: 거대소굴 해금

던전 2: 화염의 시련
 ├── 1R: 불타입 Lv.6
 ├── ...
 └── 클리어 보상: 탐색 슬롯 +1

...
```

- 한 던전 = 3~5라운드 연전
- 라운드마다 다른 이브이 선택 가능
- 같은 이브이 연속 출전 = 활동성 소모
- 패배 시 처음부터 (해당 던전만)
- 클리어한 던전 재도전 가능 (파밍 X, 기록 갱신용)

### 상성표 (간소화, 공격 유리만 적용)

| 공격 타입 | 유리 대상 |
|---|---|
| 물 | 불, 땅, 바위 |
| 불 | 풀, 얼음, 강철 |
| 풀 | 물, 땅, 바위 |
| 전기 | 물, 비행 |
| 에스퍼 | 격투, 고스트 |
| 악 | 에스퍼, 고스트 |
| 얼음 | 드래곤, 비행, 풀 |
| 격투 | 바위, 강철, 왕 |
| 고스트 | 에스퍼, 페어리 |
| 드래곤 | 드래곤, 왕, 메가 |
| 땅 | 전기, 불, 강철 |
| 바위 | 불, 비행, 얼음 |
| 비행 | 풀, 격투 |
| 강철 | 페어리, 얼음, 바위 |
| 페어리 | 드래곤, 악, 격투 |
| 왕 | 상성 없음 (기본 스탯↑) |
| 메가 | 상성 없음 (기본 스탯↑↑) |

## 2. 탐색 ↔ 전투 순환

### 보상 역할 분리

| | 탐색 보상 | 전투 보상 |
|---|---|---|
| **획득물** | 진화석, 열매, 조각 | (아이템 없음) |
| **효과** | 이브이 진화 = 전투력↑ | 새 탐색지 해금, 탐색 슬롯↑ |

### 전투 클리어 → 탐색 해금 맵

| 전투 스테이지 | 클리어 보상 |
|---|---|
| 던전 1 클리어 | 거대소굴 해금 |
| 던전 2 클리어 | 탐색 슬롯 2칸 |
| 던전 3 클리어 | 고대유적 해금 |
| 던전 4 클리어 | 깊은동굴 해금 |
| 던전 5 클리어 | 탐색 슬롯 3칸 |
| 던전 6 클리어 | 용의계곡 해금 |
| 던전 7 클리어 | 하늘산등성이 해금 |
| 최종 보스 클리어 | 엔딩 |

> 기본 8곳 탐색지는 처음부터 이용 가능
> 히든 탐색지는 전투 클리어로만 해금 (기존 조건 제거)

## 3. 탐색 시스템 (서브 콘텐츠)

### 출정 방식
```
탐색 버튼 → 장소 선택 → 이브이 선택 (1마리 이상)
  → 파견 출동 (실시간 대기)
  → 자동 귀환 + 결과 보고
```

### 탐색 규칙
- 파견 슬롯: 초기 1칸 (전투 클리어로 확장)
- 기본 이브이 + 진화 이브이 모두 파견 가능
- 진화 이브이 타입 보너스: 속성 매칭 시 아이템 확률 +25%
- 비행타입 파견 → 탐색 시간 단축
- 격투타입 파견 → 보너스 드롭 확률↑
- 친밀도 높을수록 레어 확률↑
- 활동성 높을수록 탐색 시간↓
- 파견 중 이브이는 전투 출전 불가 (운용 판단 필요)

### 탐색지 목록

#### 기본 탐색지 (처음부터 이용 가능)

| 탐색지 | 시간 | 속성 | 아이템 드롭 | 탐색 시간 |
|---|---|---|---|---|
| 호수 | 아무때나 | 물 | 물의돌, 태양조각 | 30초 |
| 화산 | 아무때나 | 불 | 불꽃의돌, 태양조각 | 30초 |
| 꽃밭 | 낮 | 페어리 | 리본, 태양조각 | 30초 |
| 숲 | 아무때나 | 풀 | 이끼바위조각, 태양/달조각 | 30초 |
| 발전소 | 아무때나 | 전기 | 천둥의돌, 태양/달조각 | 30초 |
| 설산 | 아무때나 | 얼음 | 얼음바위조각, 태양/달조각 | 45초 |
| 신전 | 낮 | 에스퍼 | 태양조각, 달조각 | 45초 |
| 폐탑 | 밤 | 악 | 달조각, 태양조각 | 45초 |

> 열매는 모든 탐색지에서 6종 균등 확률로 드롭

#### 히든 탐색지 (전투 클리어로 해금)

| 탐색지 | 시간 | 해금 조건 | 아이템 드롭 | 탐색 시간 |
|---|---|---|---|---|
| 거대소굴 | 밤 | 던전 1 클리어 | 다이버섯(희귀) | 60초 |
| 고대유적 | 밤 | 던전 3 클리어 | 이브이나이트(희귀) | 90초 |
| 깊은동굴 | 밤 | 던전 4 클리어 | 부드러운모래 | 60초 |
| 용의계곡 | 낮 | 던전 6 클리어 | 용의비늘(희귀) | 90초 |
| 하늘산등성이 | 낮 | 던전 7 클리어 | 바람깃털 | 60초 |

### 드롭 확률
- 아이템: 40% (가중치 랜덤), 타입 매칭 시 +25%
- 열매: 60% (6종 균등)
- 보장: 둘 다 안 나오면 열매 1개
- 희귀 진화석 weight 15 (기본 30)

## 4. 이브이 둥지

- 농장에 기본 이브이가 0마리일 때 소환 가능
- 비용: 열매 N개 (초반 소량)
- 기본 이브이 보유 중이면 비활성
- 진화 시 자동 이브이 추가 X → 둥지에서 수동 소환

### 진화 후 흐름
```
진화 발생 → 진화 이브이 농장 배치
          → 기본 이브이 사라짐
          → 둥지에서 새 이브이 수동 소환
```

## 7. 친밀도·활동성 시스템

### 친밀도 (기본 이브이 + 진화이브이 공통)
- 좋아하는/싫어하는 열매 각 1개 랜덤
- 반응: ♥(좋아함 +5), 💢(싫어함 -2), ⚡(보통, 스태미나+1)

| 항목 | 수치 |
|---|---|
| 시작 | 3 |
| MAX | 10 |
| 좋아하는 열매 | +5 |
| 싫어하는 열매 | -2 |
| 태양조각 (낮) | +6 |
| 달조각 (밤) | +6 |

### 활동성 (진화이브이)
- 0~50
- 활동성이 높을수록: 탐색 시간↓, 전투 스탯 보정↑, 이동 활발
- 활동성 소모: 전투 출전, 탐색 파견
- 활동성 회복: 열매 먹이기, 시간경과

### 전투·탐색에서의 역할

| 능력치 | 전투 | 탐색 |
|---|---|---|
| 친밀도 | 전투력 보정 | 레어 확률↑ |
| 활동성 | 출전 비용 | 탐색 시간↓ |

## 11. 세이브 시스템

### 저장 방식
- **localStorage** 사용 (키: `eevee_farm_save`)
- JSON 직렬화/역직렬화

### 저장 대상

| 데이터 | 설명 |
|---|---|
| currentEevee | 기본 이브이 상태 (친밀도, 좋아함/싫어함, 스태미나 등) |
| farm[] | 진화이브이 목록 (타입, 활동성, 좋아함/싫어함) |
| inventory | 보유 아이템 |
| collected | 수집한 진화형 Set |
| timeOfDay | 현재 낮/밤 |
| turnNumber | 현재 턴 |
| skipCount, totalRests, totalExplores | 누적 카운터 |
| battleProgress | 전투 클리어 현황 (던전별) |
| unlockedLocations | 해금된 탐색지 |
| expeditionSlots | 탐색 슬롯 수 |
| farmItems[] | 농장 바닥 아이템 |

### 저장 타이밍
- **자동 저장**: 진화 완료, 전투 클리어, 탐색 귀환, 휴식 시
- **수동 저장 없음** (자동만)

### 로드 타이밍
- 게임 시작 시 localStorage에 세이브 존재하면 자동 로드
- 세이브 없으면 타이틀 → 새 게임

### 타이틀 화면 변경
```
[세이브 없음]
  "터치하여 시작" → 새 게임

[세이브 있음]
  "이어하기" → 세이브 로드
  "처음부터" → 확인 팝업 → 세이브 삭제 + 새 게임
```

### 주의사항
- `collected`는 Set → Array로 변환하여 저장, 로드 시 복원
- `visitedLocations`도 Set → Array 변환
- farmItems의 물리 상태(vx, vy)는 저장하지 않아도 됨 (0으로 초기화)

## 13. 전투 스테이지 구조

### 무한 스테이지
- 끝이 없는 무한 웨이브 (적 전투력 점진적 상승)
- 스테이지 N → 적 전투력 = 기본값 × (1 + N × 0.15) 등 스케일링
- 막히면 → 탐색으로 진화체 추가 / 친밀도 올려서 돌파
- 최고 스테이지 = 플레이어 기록

### 스테이지별 탐색 해금 (고정)

| 스테이지 | 보상 |
|---|---|
| 5 클리어 | 거대소굴 해금 |
| 10 클리어 | 탐색 슬롯 2칸 |
| 15 클리어 | 고대유적 해금 |
| 20 클리어 | 깊은동굴 해금 |
| 25 클리어 | 탐색 슬롯 3칸 |
| 30 클리어 | 용의계곡 해금 |
| 35 클리어 | 하늘산등성이 해금 |

> 이후 스테이지는 해금 보상 없이 기록 갱신용

## 14. 엔딩 + 엔드게임

### 엔딩 조건
- **17종 컴플리트** → 엔딩 연출 (축하 화면 + 퍼레이드 + 통계)
- 보너스 이브이 추가 (기본 스탯 상향)

### 컴플리트 후 변경
- **친밀도 상한 해제**: MAX 10 → 무제한
- 열매를 계속 먹여서 무한 성장 가능
- 전투력도 무한 성장 → 더 높은 스테이지 도전
- 전 이브이에 기본 스탯 보정 추가

### 엔드게임 루프
```
무한 전투 → 막힘 → 친밀도 올리기 (열매 파밍)
  → 돌파 → 기록 갱신 → 리더보드 등록
  → 반복
```

## 15. 리더보드 (Google Sheets)

### 구조
- Google Apps Script 웹앱으로 배포 → REST API 역할
- 별도 서버 불필요, CORS 자유

### API
- `POST`: 점수 등록 (닉네임, 최고 스테이지, 수집 수)
- `GET`: 상위 20명 조회

### 데이터

| 컬럼 | 설명 |
|---|---|
| 닉네임 | 플레이어 입력 (3~8자, 최초 1회) |
| 최고 스테이지 | 전투 최고 기록 |
| 수집 수 | 진화체 보유 수 (/17) |
| 날짜 | 등록 시점 |

### 동작
- 전투 최고 기록 갱신 시 자동 업로드
- "랭킹" 버튼으로 리더보드 조회
- 오프라인이면 로컬 기록만 표시
- 닉네임은 localStorage에 저장

## 16. v1 → v3 마이그레이션 변경점

| 항목 | v1 (현재) | v3 (변경) |
|---|---|---|
| 게임 목표 | 이브이 17종 수집 | 17종 수집 + 무한 전투 기록 |
| 메인 콘텐츠 | 없음 (수집이 전부) | 자동 전투 (무한 스테이지) |
| 이브이 추가 | 진화 시 자동 | 둥지에서 수동 소환 |
| 탐색 | 즉시 결과, 이브이만 | 출정→대기→귀환, 모든 이브이 |
| 탐색지 해금 | 수집 조건 | 전투 스테이지 보상 |
| 열매 | 친밀도용 | 친밀도 + 활동성 + 소환 비용 |
| 활동성 | 이동속도만 | 전투 비용 + 탐색 효율 |
| 친밀도 | 진화 조건 (MAX 10) | 진화 + 전투력 (컴플리트 후 무제한) |
| 스태미나 | 탐색 소모 | 폐지 (출정으로 대체) |
| 세이브 | 없음 | localStorage 자동저장 |
| 리더보드 | 없음 | Google Sheets 연동 |
| 엔딩 | 17종 수집 | 17종 수집 (전투는 무한 계속) |
